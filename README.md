"# ComputerArchitecureTask1" 

This is part 1 of Computer Architecure Project
to design a processor in Python.

Included in this Project:
    - Signed 32 bit Input Parser w/ overflow and saturation protection
    - Conversion from:
        - Decimal -> Binary
        - Decimal -> Hexadecimal
        - Binary -> Hexadecimal
        - Binary -> Decimal
